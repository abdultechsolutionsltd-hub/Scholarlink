# ScholarLink

ScholarLink is a platform that helps students find verified scholarships, internships, and opportunities.

## Features
- Verified opportunities
- AI recommendations
- Saved opportunities
- Email reminders

## Tech Stack
- Backend: Flask + PostgreSQL
- Frontend: React + Tailwind
- AI: OpenAI API

## Deployment
- Backend: Render
- Frontend: Netlify

## Run Locally

### Backend
cd backend  
pip install -r requirements.txt  
python app.py  

### Frontend
cd frontend  
npm install  
npm start
